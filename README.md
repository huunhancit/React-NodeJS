# React-NodeJS
Project is fullstack:
  - Font-End: ReactJS, Redux
  - Backend: NodeJS, MongoDB
 
Setup project:
 - Require install nodeJS on PC
 - Install NPM 
 
Project ServerNodeJS
  - `npm install`
  - `npm run build`
  - `npm start`
  
Project ClientReactJS:
  - `npm install`
  - `npm start`
